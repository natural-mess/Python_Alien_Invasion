# Python_Alien_Invasion

Alien Invasion game using Pygame.

I use this game to learn Python from [Python Crash Course by Eric Matthes](https://ehmatthes.github.io/pcc_3e/).

## Prerequisite
* Python
* Pygame

## Game settings
Check [setting.py](./settings.py) file. 

## How to play
Checkout the repository and run the following command to start the game:

```
python alien_invasion.py
```
